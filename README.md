# Swiggy Restaurant Data Analysis Using SQL

## Project Overview
This project focuses on analyzing Swiggy restaurant data using SQL to extract business insights related to restaurant revenue, cuisine performance, customer engagement, pricing patterns, and city-wise trends.

---

## Objectives
- Analyze restaurant revenue patterns
- Identify top-performing cuisines
- Compare restaurant chains
- Perform city-wise analysis
- Use SQL window functions for ranking and analytics

---

## SQL Concepts Used
- GROUP BY
- Aggregate Functions
- Subqueries
- Window Functions
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- PARTITION BY
- ORDER BY

---

## Business Problems Solved
1. Identified restaurants generating maximum revenue.
2. Ranked restaurants city-wise and cuisine-wise.
3. Analyzed cuisine performance by revenue.
4. Compared restaurant chains based on revenue generation.
5. Performed city-wise restaurant analysis.
6. Analyzed customer engagement using rating counts.
